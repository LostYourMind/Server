    1. DB를 다른 컴퓨터에서도 접속 가능하게 권한 수정(IP값)

FastAPI 서버 관련
    1. 장바구니 기능 구현
        I.     {product 값} + "장바구니에 담아줘" or "장바구니에 추가해줘" 형식으로 클라이언트에서 전송
        II.    클라이언트 측에서 보낸 {product 값} 과 DB에 존재하는 해당 매장의 [product 값] 중 일치하는 값 추출
        III.   클라이언트에게 추출된 (product 값)을 전송
        IIII.  클라이언트는 FastAPI 서버가 보낸 (product 값)을 처리해서 장바구니에 추가 후 리렌더링

    2. 음성 인식 기능 알고리즘 구현 
        I.     기본적으로 AI가 하는 모든 [Text] 값은 TTS를 사용해서 읽어 줄 수 있게 -> 구현 완료 / 테스트 필요
        II.    클라이언트 -> FastAPI 로 사용자가 말하는 모든 음성 데이터를 어떤 방식으로 넘길 지 구성
        III.   넘어온 데이터를 기반으로 FastAPI 서버 구조 변경

    3. 인기 메뉴 추천 알고리즘 구현
        I.     메뉴가 주문되면 해당 메뉴에 증감값 부여
        II.    "인기메뉴 추천해줘" or "인기메뉴는 뭐가 있을까?" 와 비슷한 형식으로 클라이언트에서 전송
        III.   구분 기준 선정
        IIII.  [product]에 들어있는 증감값들 중 가장 높은 5개의 메뉴 선정(임의/수정가능)
        IIIII. 해당 추출된 [product] 값 클라이언트에게 return
